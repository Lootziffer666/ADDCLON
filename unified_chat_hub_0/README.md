# ChatHub Sidebar (MVP)

Chrome-Erweiterung für **ChatGPT + Gemini** mit:

- Prompt-Bibliothek/Snippets (CRUD)
- Schnellzugriff unter dem Eingabefeld
- Verlauf + Suche
- Token-/Kosten-Schätzung
- Export/Import (JSON)
- Antwort-Tools (Copy/Clean/Translate/Summarize)
- Hotkey: `Ctrl+Shift+Y`

## Hinweis zu Variablen (`{{topic}}`)
Variablen sind im MVP nur als Platzhalter im Prompt-Text sichtbar, aber noch ohne gesonderten Eingabedialog.

## Installation
1. `chrome://extensions` öffnen
2. Entwicklermodus aktivieren
3. "Entpackte Erweiterung laden" → diesen Ordner wählen
